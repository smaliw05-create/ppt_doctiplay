# Doctiplay Pitch Deck

A premium, interactive scrollytelling deck for **Doctiplay**.

## Features
- **Dynamic Scroll Animations**: Reveal effects and smooth snapping.
- **Premium Aesthetics**: Dark theme, glassmorphism, and modern typography.
- **Responsive Design**: Works on tablets and desktops.
- **Navigation Controls**: Progress bar, keyboard shortcuts (`Arrows`, `P` for fullscreen).

## How to Run it locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at the provided localhost link.

## Deployment

This project is ready to be deployed on **GitHub Pages**, **Vercel**, or **Netlify**.
